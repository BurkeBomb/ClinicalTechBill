
# IONM Billing Simulator

A **React + TypeScript** application for simulating **Intraoperative Neurophysiological Monitoring (IONM) billing**, integrated with **Firebase Authentication** and **Firestore** for real-time case management.

---

## ✅ Features
- **User Authentication**:
  - Email/Password Sign-In & Registration
  - Anonymous Sign-In fallback
- **Case Management**:
  - Create new cases with patient details
  - Upload placeholder images
- **Real-Time Monitoring**:
  - Start, Pause, Resume, and End monitoring sessions
  - Track elapsed and billable time
- **Billing Preview**:
  - Calculate time-based codes (137/139)
  - Generate claim summary with tariff codes
- **Responsive UI**:
  - Built with **TailwindCSS**

---

## ✅ Tech Stack
- React (TypeScript)
- Firebase (Auth + Firestore)
- React Hook Form + Zod (Validation)
- TailwindCSS (Styling)
- GitHub Actions (CI/CD)
